# Financial Dashboard Website (Clone)

This project is a replica of the provided Financial Dashboard reference website.  
It was developed using HTML, CSS, and Bootstrap to closely match the layout, styling, and responsiveness of the original site.

## Live Demo
https://jothiga18.github.io/fintech-assingment/

## Reference Website
https://financialdash.ccbp.tech/

## Tech Stack
- HTML
- CSS
- Bootstrap

## Description
The goal of this project was to recreate the overall user interface and experience of a real financial dashboard.  
All sections including dashboard cards, navigation layout, and transactions view were carefully replicated with attention to spacing, typography, colors, and responsiveness.

The code is structured and commented for clarity, making it easy to understand different components and sections.
